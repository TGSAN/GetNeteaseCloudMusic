# GetNeteaseCloudMusic

### Introduction

You can use this Python script to parse the Netease Cloud Music download address, even if the hided music

### Usage

```shell
$ python 163.py <url>
```

example:

```shell
$ python 163.py "https://music.163.com/#/song?id=441116341"
```

or

```shell
$ python 163.py "https://music.163.com/#/song?id=441116341"
```
